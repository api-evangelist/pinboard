# PinboardThis is a repo for managing machine readable artifacts for the [Pinboard APIs](http://www.pinboard.in). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Pinboard APIs](http://www.pinboard.in).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.pinboard.in)- [Developer](http://www.pinboard.in)- [Twitter](https://twitter.com/pinboard)- [GitHub](https://github.com/pinboard)The [APIs.json](https://github.com/api-evangelist/pinboard/blob/master/apis.json) index provides links to all the relevant links for the Pinboard APIs, including the OpenAPI and Postman if they are present.
